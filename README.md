# UndeRRated

## Creadores

Somos tres creadores: [Adrian Ontiveros](<adrian.ontiveros.7e4@itb.cat>), [Agustin Garibotto](<agustin.garibotto.7e6@itb.cat>) y [Olav Martos](<olav.martos.7e4@itb.cat>)

## Información del proyecto

UndeRRated es un proyecto escolar realizado con el motor de videojuegos Unity

<img src=".\UndeRRated\Readme\LogoUnity.png" alt="IconoUnity" style="width:100px"/>

Este proyecto se trata de realizar un Endless Runner con nuestro propio estilo.
Siguiendo unas rúbricas específicas.
Nosotros hemos decidido desarrollar nuestro juego en 3D.
